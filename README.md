# 🏆 Tu Tapa Fixture

¡Diseñá y Descargá tu Fixture Mundial 2026 Gratis! Una herramienta profesional, intuitiva y lista para imprimir en formato A4.

![Tu Tapa Fixture Banner](og_image.jpg)

## 🌟 Descripción

**Tu Tapa Fixture** es una aplicación web diseñada para que cualquier persona, comercio o empresa pueda crear su propio fixture del mundial personalizado. El sitio permite generar un archivo PDF de alta calidad, optimizado para impresión en hoja A4, con un diseño moderno y minimalista.

Ideal para:
- 🏪 **Negocios**: Agrega tu marca y colores para repartir a tus clientes.
- 🎁 **Regalos**: Crea un fixture único para tus amigos o familia.
- ⚽ **Fanáticos**: Tené siempre a mano el calendario de partidos con estilo.

---

## 🚀 Características Principales

- **🖼️ Creador de Portadas Inteligente**: Subí tu propia imagen (3:4) para la tapa. ¿No tenés una? La app te ofrece un **Prompt de IA** optimizado para que generes una imagen profesional en segundos.
- **🎨 Personalización de Colores**: Cambiá los filtros de las tarjetas del fixture en tiempo real para que coincidan con tu marca o gustos.
- **📄 Exportación Dual en PDF**:
    - **Modo Plegable**: Configurado para que al doblar la hoja A4 obtengas un librito de bolsillo.
    - **Modo Normal**: Una disposición estándar para lectura directa.
- **📱 Optimizado para Móviles**: Diseñá desde cualquier dispositivo. La previsualización pesada se oculta en celulares para mejorar el rendimiento, pero la descarga sigue siendo perfecta.
- **🛠️ Herramientas Profesionales**: Integración con **Fabric.js** para la edición de la tapa y **jsPDF** para la generación del documento final.

---

## 🛠️ Tecnologías Utilizadas

El proyecto está construido con tecnologías web modernas sin dependencias pesadas de frameworks, priorizando la velocidad y la compatibilidad:

- **HTML5 & CSS3**: Diseño responsive con variables CSS y Glassmorphism.
- **JavaScript (Vanilla)**: Lógica de la aplicación y manipulación del DOM.
- **[Fabric.js](http://fabricjs.com/)**: Potente motor de Canvas para la edición de la portada.
- **[jsPDF](https://github.com/parallax/jsPDF)**: Generación de documentos PDF en el lado del cliente.
- **[html2canvas](https://html2canvas.hertzen.com/)**: Captura de elementos visuales para la composición del PDF.

---

## 📖 Cómo Usar

1.  **Paso 1: Crear Tapa**: Sube una imagen o usa la guía de IA para obtener un prompt creativo.
2.  **Paso 2: Colores**: Ajusta los 3 selectores de color para darle vida a las fases del fixture.
3.  **Paso 3: Exportar**: Elegí el formato que prefieras (Plegable o Normal) y descargá tu PDF.
4.  **Paso 4: ¡Imprimir!**: Si elegiste el modo plegable, seguí el tutorial de doblado incluido en la web.

## 🔧 Instalación Local

Si querés ejecutar el proyecto en tu máquina local:

1. Cloná el repositorio:
   ```bash
   git clone https://github.com/Lautarof25/TuTapaFixture.git
   ```
2. Navegá a la carpeta:
   ```bash
   cd TuTapaFixture
   ```
3. Abrí el archivo `index.html` en tu navegador favorito o usá una extensión de Live Server.
